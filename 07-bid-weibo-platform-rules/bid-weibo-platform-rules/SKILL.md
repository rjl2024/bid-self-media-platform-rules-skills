# Weibo Platform Rules

Use this skill for Weibo posts, image/video posts, long posts, topic operations, and public-opinion-sensitive content.

## Workflow

1. Identify content type: short post, long post, image post, video, live, super topic, commercial campaign.
2. Read `references/weibo.md`.
3. Check community risks: public events, rumors, hate/abuse, privacy, marketing, off-platform diversion, and rights.
4. Verify upload limits in Weibo creator/upload backend when file specs matter.

## Response Pattern

Give recommended exports, text and hashtag risks, rights checks, and backend-verification items.
