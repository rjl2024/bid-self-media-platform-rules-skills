安装步骤

  1. 确认交付包位置
    I:\闲鱼发布产品库\063005自媒体平台规则skill发布包
  2. 创建本地 Skills 目录
    $dstRoot = "$env:USERPROFILE\.codex\skills"
New-Item -ItemType Directory -Force -Path $dstRoot | Out-Null
  3. 一次安装全部平台 Skill
    $srcRoot = "I:\闲鱼发布产品库\063005自媒体平台规则skill发布包"
$dstRoot = "$env:USERPROFILE\.codex\skills"
Get-ChildItem -Directory $srcRoot | ForEach-Object {
  Get-ChildItem -Directory $_.FullName | Where-Object { Test-Path (Join-Path $_.FullName "SKILL.md") } | ForEach-Object {
    $dst = Join-Path $dstRoot $_.Name
    if (Test-Path $dst) { Rename-Item $dst "$($_.Name)-backup-$(Get-Date -Format yyyyMMddHHmmss)" }
    Copy-Item $_.FullName $dst -Recurse
  }
}
  4. 重启客户端

    复制完成后重启 Codex，让它重新扫描技能目录。之后可直接用 $bid-douyin-platform-rules、$bid-xiaohongshu-platform-rules 等名称调用。

使用方法

发布前审稿

检查标题、封面、正文、视频信息和潜在违规点。

素材导出建议

按目标平台给出封面、图片、视频推荐尺寸。

跨平台改稿

把内容改成另一个平台更合适的标题、图文和风险提示。

常见问题

为什么有些平台没有写死上传大小和时长？

这些限制经常在创作者后台按账号权限、端版本、功能灰度变化。Skill 会保留推荐规格和官方入口，对不稳定字段要求后台复核。

zip 怎么用？

zip 是方便分发的安装包。实际安装时解压后，把里面的同名 Skill 文件夹复制到 %USERPROFILE%\.codex\skills。
