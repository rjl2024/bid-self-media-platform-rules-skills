# Kuaishou Platform Rules

Use this skill for Kuaishou China content. Kuaishou short-video, live, and e-commerce rules can differ; identify the route before giving specs.

1. Identify route: normal post, live, shop/e-commerce, local life, ad/commercial content.
2. Read `references/kuaishou.md`.
3. Check media specs, product claims, live behavior, traffic diversion, rights, and AIGC disclosure.
4. Mark account/backend-specific limits as verification items.

## Response Pattern

Give recommended exports, hard/known restrictions, risk fixes, and backend-verification items.
